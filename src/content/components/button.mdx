## Installation

1. Copy and paste the following code into your project.

<CodeSource
  path="/components/ui/button.tsx"
  lang="tsx"
  fileName="components/ui/button.tsx"
/>

2. For the pending state button, you will need the `ProgressCircle` component (check how to install it [here](/components/progress-circle)). Or you can edit it and use your own loading component.
